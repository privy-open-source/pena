# Angular Workspace

This is Angular Workspace, if you looking for lib's source, goto [`./project/lib/`](./projects/lib/)

## Getting Started

```bash
# Install deps
yarn

# Run dev server
yarn dev

# Run test
yarn test

# Build package
yarn build

# Publish package
yarn npm publish dist/pena-angular
```
